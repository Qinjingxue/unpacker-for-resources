# unpacker-for-resources
Chatgpt5.1写的一个windows下自动循环解压目录下（不包含子目录）文件的小脚本

## 依赖
- python>=3.6
- 环境变量中包含7z.exe

## 功能
- 自动撞库，如果目录下有多个压缩包包含多个解压密码，自动按一开始输入的解压密码一个一个试
- 魔数识别压缩包，不依赖扩展名
- 循环解压嵌套压缩包：压缩包解压后会自动识别解压后的目录下（包含子目录）还有没有压缩包，有的话继续解压
- 支持分卷压缩
- 多线程解压，根据IO来自动判断开多少线程，防止开过多线程导致IO瓶颈，7z忙等
- 解压完成后自动把解压完成的所有压缩包扔到回收站，同时压平单子嵌套目录
- 解压失败的压缩包自动重试，同时输出错误日志
- 磁盘不足时自动清除最早解压的压缩包，防止磁盘满解压失败

## 文件说明
- 自动撞库解压带IO检测和分卷处理.py 包含上述所有功能
- GUI版本是根据“自动撞库解压带IO检测和分卷处理.py ”添加简单GUI后的版本
- 自动撞库解压.py 不包含IO检测和分卷处理，电脑不够好的不要轻易尝试，很容易卡死，反之电脑足够好的用这个解压最快
- 自动撞库解压带IO检测.py 不带分卷处理功能的中间版本
